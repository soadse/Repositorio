Sobre mim

Meu nome é Rodrigo e sou apaixonado por tecnologia. Meu primeiro contato com a programação foi em 2022, e desde então, não parei mais de aprender. Hoje, já entendo bastante sobre tecnologias voltadas para o Front-end.

Adoro "brincar" de criar interfaces, seja para projetos pessoais ou para ajudar amigos. Gosto de experimentar e ver o que posso construir com as ferramentas que conheço, como HTML, CSS e JavaScript. Criar essas interfaces é uma forma de unir minha curiosidade técnica com a diversão de ver algo ganhar vida na tela. A cada novo projeto, aprendo algo novo e me desafio a melhorar minhas habilidades.

Programar se tornou uma parte importante da minha vida, e estou sempre buscando novas oportunidades para me aprimorar e descobrir mais sobre o mundo do desenvolvimento web.

Habilidades

Passa o mouse por cima de alguma habilidade para ler a descrição

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página.

O Python é uma linguagem de programação amplamente usada em aplicações da Web, desenvolvimento de software, ciência de dados e machine learning (ML). Os desenvolvedores usam o Python porque é eficiente e fácil de aprender e pode ser executada em muitas plataformas diferentes.

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. É atualmente um dos sistemas de gerenciamento de bancos de dados mais populares da Oracle Corporation, com mais de 10 milhões de instalações pelo mundo.

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.
